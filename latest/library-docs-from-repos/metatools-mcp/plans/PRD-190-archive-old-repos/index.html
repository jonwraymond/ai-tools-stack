    PRD-190: Archive Old Repos¶

    Phase: 9 - Cleanup Priority: High Effort: 2 hours Dependencies: PRD-180 Status: Done (2026-02-01)

    Objective¶

    Archive all 13 standalone repositories that have been consolidated.

    Repositories to Archive¶

    Repository Migrated To
    toolmodel toolfoundation/model
    tooladapter toolfoundation/adapter
    toolindex tooldiscovery/index
    toolsearch tooldiscovery/search
    toolsemantic tooldiscovery/semantic
    tooldocs tooldiscovery/tooldoc

    toolset toolcompose/set
    toolskill toolcompose/skill

    Tasks¶

    Task 1: Add Deprecation Notices¶

    For each repo, update README.md:

    # ⚠️ DEPRECATED

This repository has been archived. The code has been migrated to:

**[toolfoundation](https://github.com/jonwraymond/toolfoundation)** (model package)

## Migration

Update your imports:

```go
// Before
import "github.com/jonwraymond/toolmodel"

// After
import "github.com/jonwraymond/toolfoundation/model"

    See the stack migration guide for details: https://jonwraymond.github.io/ai-tools-stack/operations/migration-guide/ 

    ### Task 2: Create Migration Guides

**File: MIGRATION.md** (per repo)

```markdown
# Migration Guide

## Import Changes

| Old | New |
|-----|-----|
| `github.com/jonwraymond/toolmodel` | `github.com/jonwraymond/toolfoundation/model` |

## Breaking Changes

- Package name changed from `toolmodel` to `model`
- All functionality preserved

## Steps

1. Update go.mod:
   ```bash
go get github.com/jonwraymond/toolfoundation@latest
   ```

2. Update imports:
   ```bash
sed -i 's|github.com/jonwraymond/toolmodel|github.com/jonwraymond/toolfoundation/model|g' *.go
   ```

3. Remove old dependency:
   ```bash
   go mod tidy
   ```

    Task 3: Archive Repositories¶

    REPOS=(
  toolmodel
  tooladapter
  toolindex
  toolsearch
  toolsemantic
  tooldocs
  toolrun
  toolruntime
  toolcode
  toolset
  toolskill
  toolobserve
  toolcache
)

for repo in "${REPOS[@]}"; do
  echo "Archiving jonwraymond/$repo..."

  # Update README with deprecation notice
  # (done manually per repo with appropriate redirect)

  # Archive the repository
  gh repo archive "jonwraymond/$repo" --yes

  echo "✓ $repo archived"
done

    Task 4: Verify Archives¶

    # List archived repos
gh repo list jonwraymond --archived --limit 20

# Verify each is read-only
for repo in toolmodel tooladapter toolindex toolsearch; do
  gh repo view "jonwraymond/$repo" --json isArchived -q '.isArchived'
done
